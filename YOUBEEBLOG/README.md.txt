# Full-Stack Login & Posts System (PHP / CakePHP Style)

This project is a **full-stack web application** demonstrating user authentication and post management, structured in a way similar to **MVC frameworks (like CakePHP)**.  
It showcases my skills in both **front-end and back-end development**, as well as database integration.

---

## 🚀 Features
- User registration (Sign Up)  
- User authentication (Login / Logout)  
- Creating and managing posts  
- Reusable front-end components (navbar, footer, etc.)

---

## 🛠 Technologies Used
- **Front-End:** HTML, CSS, JavaScript  
- **Back-End:** PHP (structured following MVC principles)  
- **Database:** MySQL (via phpMyAdmin in XAMPP)

---

## 📂 Project Structure
- `/actions` → backend PHP logic (controllers in MVC terms)  
- `/components` → reusable front-end parts (views in MVC terms)  
- `/database` → SQL export (`youbeeblog1.sql`) for easy setup  
- Other `.php` files → main pages (home, login, signup, posts, etc.)  

> This structure mimics **CakePHP/MVC**, making it easier to scale in the future.

---

## ⚙️ How to Run the Project
1. Install **XAMPP** (or any PHP & MySQL server).  
2. Copy the project folder into the `htdocs` directory.  
3. Start **Apache** and **MySQL** in XAMPP.  
4. Open [http://localhost/phpmyadmin](http://localhost/phpmyadmin).  
5. Create a new database (e.g., `myproject`).  
6. Import the SQL file from `/database/youbeeblog1.sql`.  
7. Open your browser:  

http://localhost/YOUBEEBLOG
